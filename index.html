Что такое PostgreSQL?
СУБД open‑source

PostgreSQL — мощная объектно‑реляционная СУБД.
Подходит для веб‑приложений, аналитики и сложных запросов.

Подключение (psql)

Формат:

-- из терминала
psql -h host -p 5432 -U user -d dbname

Если база локально и пользователь по умолчанию:
psql -d postgres

Базовый CRUD (SELECT / INSERT / UPDATE / DELETE)

Выборка (SELECT):

-- все колонки
SELECT * FROM users;

-- конкретные колонки + фильтр
SELECT id, email FROM users
  WHERE active = true;

Добавление (INSERT):

INSERT INTO users (email, password)
VALUES ('test@example.com', 'secret')
RETURNING id;
Создание таблицы (CREATE TABLE)
serial / bigserial
primary key
CREATE TABLE users (
  id SERIAL PRIMARY KEY,
  email TEXT NOT NULL UNIQUE,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

Фильтры и сортировка

WHERE, ORDER BY, LIMIT:

SELECT id, email
FROM users
WHERE created_at > '2024-01-01'
ORDER BY created_at DESC
LIMIT 10 OFFSET 0;
JOIN (соединение таблиц)

INNER JOIN пример:

SELECT u.email, o.total
FROM users u
JOIN orders o
  ON o.user_id = u.id;

Другие варианты: LEFT JOIN, RIGHT JOIN, FULL JOIN.

Индексы

Индексы ускоряют поиск по колонкам, но замедляют вставку/обновление.
Создавай индексы по часто используемым фильтрам и связям.

CREATE INDEX idx_users_email
ON users (email);
Типичные команды администратора

Создание БД и пользователя:

CREATE DATABASE app_db;
CREATE USER app_user WITH PASSWORD 'secret';
GRANT ALL PRIVILEGES ON DATABASE app_db
TO app_user;

Мини‑чеклист для новичка
  • Установи PostgreSQL и клиент psql.
  • Создай отдельную базу и пользователя под проект.
  • Добавь первую таблицу и потренируйся в CRUD.
